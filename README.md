<h1 align="center"> kaokun </h1>
<div align="center">

Randomize a **Kaomoji** (**顔文字**) with custom seeds

Currently featuring 3000~ kaomoji

**(☞ﾟヮﾟ)☞**

</div>

**npm**

```
npm install kaokun
```
**yarn**

```
yarn add kaokun
```
